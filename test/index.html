_(selector)/__(selector)

HTML文書の要素を選択するメソッド。

jQueryで言うところの$()に相当する。

また、内部的にはdocument.querySelector()やdocument.querySelectorAll()を使用しているため、動作速度は比較的遅い。

alert(`以下、_()の結果。\n${_('div').innerText}\n以下、__()の結果。${__('div')[0].innerText}`);
動作させる

_newEl(object)

新しい要素を作成するメソッド。

document.createElement()を使用するため、DOMには追加されない。

オブジェクト：

{
  el: '<要素の名前>',
  attribute: [
    ['<属性の名前>','<属性の値>']
  ],
  value: '<要素の値>',
  inner: '<要素の内容(innerHTML)>'
}

以下、テスト用のコード。

var element=_newEl({
  el: 'p',
  inner: 'new <span><p</span>> element.',
  attribute: [
    ['style','font-size; 15px;'],
    ['onclick','alert(\'This element is \"p\" element\')']
  ]
})
_('div:nth-child(3)').appendChild(element);
動作させる

_notNull(variable)

変数の内容があるかを確認するメソッド。

var forNotNull='notnull';
alert(_notNull(forNotNull));
動作させる

_attr(element array)

HTML文書内の要素の属性を変更するメソッド。

初期状態では属性を追加するが、引数の配列内に['overWrite',true]を追加すると上書きできる。結局は実装できず諦めました。

また、引数は要素および配列でなければいけません。テキストや、連想配列では正しく動作するわけがありません。

_attr(_('div:nth-child(5)>span[style]'),[
  ['style','size: 15px;'],
  ['onclick','alert(\'This element is \"p\" element\')']
])
動作させる クリックしてください
